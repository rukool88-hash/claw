# brain-vault

个人知识管理 + AI 创作机器人（5 层架构）。

- 第一层 Capture：手机录音 / RSS / 书摘 / 会议录音
- 第二层 Process：n8n + Whisper（云+本地）
- 第三层 Storage：本仓库（Obsidian Vault）
- 第四层 AI Engine：Claude Code（由 [CLAUDE.md](CLAUDE.md) 驱动）
- 第五层 Output：CEP / 白皮书 / 视频脚本 / 博客
